# NeuroLearn

NeuroLearn is a modern online learning platform offering curated courses, bite-sized lessons, and skill-based programs (Artificial Intilligence and Research). Learners can explore interactive lessons, watch video content, complete hands-on exercises, and enroll in guided tracks designed to build practical, job-ready skills.

## Purpose

Provide a focused environment to prototype UI components, practice React patterns, and demo micro-interactions with real packages and small datasets.

## Key Features

- Minimal, modern Next.js app structure
- Responsive layouts and accessible components
- Form handling with `react-hook-form`
- Carousel / media interactions with `swiper`
- Smooth marquees with `react-fast-marquee`
- Toast notifications with `react-toastify`
- Local prototype API via `json-server`

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS + DaisyUI
- MongoDB adapter available for integration
- Better Auth
